Models Used

  • BLIP: Base model for image understanding and caption generation
  • Kosmos-2: Advanced vision-language model for grounded caption generation
  • CLIP: Used for caption ranking and validation
  • GPT-2: Enhances caption fluency and natural language generation
  • SentenceBERT: Used for semantic similarity and caption fusion

Fusion Techniques

  • CLIP Fusion: Ranks and selects best captions using vision-language alignment
  • Naive Fusion: Combines captions using semantic similarity
  • VAE Fusion: Uses GPT-2 for enhanced caption generation
  • Multi-Model Ensemble: Combines strengths of different models
  • Grounded Generation: Ensures captions are visually grounded

Evaluation Metrics

  • Vision-Language: CLIP score for image-text alignment
  • Semantic: SentenceBERT similarity scores
  • Fluency: GPT-2 perplexity measures
  • Grounding: Kosmos-2 visual grounding accuracy
  • Ensemble: Combined model confidence scores

About the Project

This project aims to build an image captioning system using deep learning techniques. The goal is to generate accurate captions for uploaded images using trained models.

Project Requirements

The project requires a deep learning model, pre-trained dataset, and a web interface. Technologies used include Python, Flask, TensorFlow, and HTML/CSS for frontend.
